# Instagram-MERN-fullstackclone
MERN based Instagram fullstack clone 

- For this insta clone i have used following main techstack-
  - Reactjs
  - Tailwindcss
  - Nodejs
  - Expressjs
  - Mongodb
 
- I have used following dependent library-
  - Redux
  - Redux-toolkit
  - React-loading-library
  - Jwt
  - React-query
  - React-router-dom
